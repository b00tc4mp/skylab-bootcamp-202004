Juego del Pasapalabra

El juego pasapalabra consiste en acertar las veinticinco palabras del rosco, cada una de las cuales se corresponde con una letra del rosco para la que se ofrece una definición relativa a un concepto Físico. Las palavras devem ser escritas sin acentos.

  1. Cuando escriba la respuesta a la pergunta deve presionar la tecla "ok" o apertar "enter".
  2. Se acerta la palavra la letra del rosco se pondrá de color verde.
  3. Se falla la palavra la letra del rosco se pondrá de color rojo.
  4. Puede tambien pasar la palabra con la tecla "pasapalabra" o apertando "espacio".
  5. Para completar el rosco se dispone de un tiempo de 150 segundos.
  6. El juego termina cuando se haya terminado el tiempo o se haya completado el rosco.
  7. Para reiniciar el juego puede presionar la tecla "restart" o aprertando "esc".
  8. Para empezar a jugar presione la tecla "jugar".
  9. Introduce tu nombre por favor !

Buena Suerte!!!
