¿Qué es Flexbox?

Es un método que nos ayuda a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación de los diferentes elementos del DOM a través de nuestra hoja de estilos.

Contexto:

Para usar flexbox se trabaja en conjunto con el contenedor y sus hijos. No hay forma de manipular directamente un elemento con flexbox sin crear antes su contexto.

Propiedades de Flexbox

Flexbox se divide en dos grupos de propiedades:

  • Las propiedades para los elementos del flex-container
  • Propiedades para los elementos de los flex-items

Activar Flexbox:

flex container 
/* ----- SINTAX ----- */
display: flex;

# Flex direction:

flex container

Nos permite modificar la dirección de los elementos. Según el valor indicado, el main axis será:

  • row: (por defectro) horizontal, izquierda a derecha
  • row-reverse: horizontal, derecha a izquierda
  • column: vertical, de arriba hacia abajo
  • column-reverse: vertical, de abajo hacia arriba
/* ----- SINTAX ----- */
flex-direction: row;

# Justify content:

relativo al main axis flex container

Alinea horizontalmente o verticalmente los elementos dependiendo de la orientación marcada por flex-direction:

  • flex-start: (por defecto) al inicio del main axis
  • flex-end: al final del main axis
  • center: al medio del main axis
  • space-around: calcula el espacio libre y lo distribuye alrededor de los flex-items
  • space-between: calcula el espacio libre y lo distribuye entre los flex-items
/* ----- SINTAX ----- */
justify-content: flex-start;

# Align items:

relativo al cross axis flex container

Nos permite alinear los flex-items respecto al cross axis. Lo utilizamos cuando tenemos los elementos en una sola linea.

  • stretch:(default) los flex-items rellenan todo el espacio disponible en el cross axis (si no se ha definido su dimensión)
  • flex-start: al inicio del cross axis
  • center: al medio del main axis
  • space-around: calcula el espacio libre y lo distribuye alrededor de los flex-items
  • space-between: calcula el espacio libre y lo distribuye entre los flex-items

Nuevamente, recuerda que aunque de manera predeterminada, el cross axis es vertical, podría ser también horizontal, si así lo definimos en flex-direction. Así que también en este caso, es un error decir que align-items se usa para alineación vertical.

/* ----- SINTAX ----- */
align-items: stretch;

# Flex wrap:

flex container

Nos permite meter en la misma caja los elementos que deberían ocupar varias lineas. De esta manera los elementos se irán adecuando. dependiendo el tamaño de la pantalla. Podemos utilizar los siguientes valores:

  • no-wrap: si no cabe se van adecuando abajo
  • wrap cada uno de los elementos tiene un ancho dinámico
  • wrap-reverse: lo mismo pero los elemento se ordenan a la inversa
/* ----- SINTAX ----- */
flex-wrap: wrap-reverse;

# Align content:

relativo al cross axis flex container

Con la propiedad align-content podemos distribuir el espacio disponible entre los elementos respecto al cross size:

  • flex-start: te ajusta los elementos arriba
  • flex-end: te ajusta os elementos abajo
  • center: te ajusta los elementos en el centro
  • space-around: ajusta los valores arriba y abajo dejando el espacio sobrante en el centro
  • space-between: ajusta los valores arriba y abajo dejando el espacio sobrante en el centro y dejando márgenes arriba y abajo
  • stretch: (por defecto) estira las lineas para ocupar todo el contenedor

Actua de la misma manera que el align-items, pero en este caso lo utilizaremos cuando tenemos más de una línea.

/* ----- SINTAX ----- */
flex-wrap: wrap;
align-content: space-between;

# Align self:

flex item

Nos permite alinear un flex-item concreto.

  • flex-start: Pega el elemento arriba
  • flex-end: pega el elemento abajo
  • center: sitúa el elemento en el centro
  • baseline: alinea el flex-item con el contenido del mismo
  • stretch: el flex-item rellena todo el espacio vertical, muy útil por ejemplo para un aside. 
/* ----- SINTAX ----- */
align-self: #;

# Flex basis:

shorthand: flex flex item

La propiedad flex-basis, como su nombre lo indica, define el main size base (tamaño base principal) para un flex-item. Cuando hablamos de main size base no especifica que necesariamente será su tamaño al dibujarse por el navegador, pero que será un punto de partida para calcular el tamaño final. En otras palabras, si el main axis es horizontal (predeterminado), flex-basis será equivalente a width; y si el main axis es vertical, flex-basis será equivalente a height. Ten en cuenta que el tamaño definido por flex-basis es, como su nombre lo dice, el tamaño base. Es decir, que podrá variar (crecer o encogerse), según los valores de flex-grow y flex-shrink que veremos más adelante. Importante:

  • flex-basis siempre gana sobre el valor de width o height
  • Si no se define el valor de flex-basis o se establece en auto, se tomará en cuenta el valor de width o height según sea el caso
  • Si no se define un valor para flex-basis y tampoco se especifica el tamaño por width o height, se definirá el main-size según su contenido
/* ----- SINTAX ----- */
flex-basis: #;
